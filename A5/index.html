How to create and instance with AWS EC2 and deploy rssParser.

Step 1: Navigate to the Amazon Web Services EC2 home page.

Step 2: Click on the orange Launch Instance button.

Step 3: Select which Amazon Machine Image you want to use.

Step 4: Select the t2.micro option.

Step 5: Click Launch in the lower right hand of the screen.

Step 6: Create and download a Key Pair for the instance.

Step 7: Make sure you see that your instance has launched.

Step 8: Make sure that your instance has passed the two status checks.

Step 9: To connect to your instance choose the EC2 Instance Connect and press Connect.

Step 10: Now you should see a console for your instance.

Step 11: To copy the rssParser into your instance navigate to your local bash console and type in
scp -i (yourpairkeyfilename).pem rssParser ec2-54-193-65-123.us-west-1.compute.amazonaws.com:services

Step 12: Now you are ready to type node server.js into your instance console and start the server for rssParser.
